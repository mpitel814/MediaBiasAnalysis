# MediaBiasAnalysis 
## Project Description
This is a final project for INFO-664. It analyzes sentiment and common words used in news headlines on a particular topic across the political spectrum.  
## Rationale Statement 
I was curious to know how the language use and sentiment of news article on a particular contentious differed depending on whether they were published by a left or right leaning publication.
## Workflow 
1. I used requests to return a large dataset of articles from the MediaStack API
2. Using OpenRefine, I filtered by dataset to only include articles from NY Times, CNN, Fox News, Daily Mail.
3. I then used NLTK to analyze sentiment of article descriptions for each
4. I used XXX to count word frequency
5. Used visualzation to create a map. 
## Further Uses 
The dataset can be used for further analysis.
